사용법

0. 내 캐릭터 정보 공개 설정

메이플스토리 공식 홈페이지 로그인 > 마이메이플 > 캐릭터 정보 더보기 
  > 내 캐릭터 정보 공개 설정 > 메이플 ID > 월드 > 캐릭터 정보 > 저장

필요한 캐릭터 정보는 기본정보, 장비, 길드임
순서1, 순서2, 순서3 사진 참고

1. Maple_info 또는 Maple_info.exe 실행

2. 캐릭터 이름 입력 후 프로필생성 클릭
     유니온 펫장비 부분은 나중에 추가할 예정
     입력해도 결과에는 아무 영향 없으니 입력할 필요 x

3. 왼쪽 진행바확인
    1에서 9까지 진행된다음 fin이 출력될 것
    fin이 출력되면 Maple_info 폴더 확인
    입력한 캐릭터 이름의 폴더가 생성되어 있을 것
    폴더에는 자신의 캐릭터의 세부 장비스탯들과 이미지가 저장됨

4. 진행이 완료됐으면 장비스탯확인, 흑우등급확인 각각 클릭해보면 됨
